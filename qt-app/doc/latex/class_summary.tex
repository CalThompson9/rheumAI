\doxysubsection{Detailed Description}
Represents a summary with various sections. 

This class provides methods to manage the interval history, physical examination, current status, and plan sections of a summary. It includes methods to get and set these sections, as well as a method to clear all sections. \begin{DoxyAuthor}{Author}
