\doxysubsection{Detailed Description}
A dialog for editing patient information. 

This class provides a user interface for editing various patient details including first name, last name, date of birth, health card number, email, phone number, address, postal code, province, and country. The dialog is initialized with a logo and provides methods to set and get the values entered in the fields. \begin{DoxyAuthor}{Author}
