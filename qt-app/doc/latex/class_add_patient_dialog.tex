\doxysubsection{Detailed Description}
Dialog for adding a new patient. 

This class provides a dialog interface for entering patient information. It includes fields for first name, last name, date of birth, health card, email, phone number, address, postal code, province, and country. It also includes validation to ensure all fields are filled before submission. \begin{DoxyNote}{Note}
The dialog is modal, meaning it blocks input to other windows until closed. 
\end{DoxyNote}
