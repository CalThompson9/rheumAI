\doxysubsection{Detailed Description}
Abstract base class for formatting summary layouts. 

This class provides an interface for all summary layout formats. It defines a method to generate the layout and a method to format bold text. The class also provides a method to clear the layout and a method to add sections. The formatting of the summary is done using HTML tags. The class is designed to be inherited by specific formatters that implement the generate\+Layout method. The class uses the Strategy pattern to allow for different formatting strategies. \begin{DoxyAuthor}{Author}
