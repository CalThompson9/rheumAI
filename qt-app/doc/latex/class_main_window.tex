\doxysubsection{Detailed Description}
Main application window for the program. 

This class manages the main window of the application, including the UI elements, patient management, and summary generation. It handles user interactions, such as adding, editing, and deleting patients, as well as generating and displaying summaries based on the patient\textquotesingle{}s transcript. The class also manages the layout of the summary display and provides functionality to switch between different summary formats. It connects to the \doxylink{class_l_l_m_client}{LLMClient} for generating summaries and handles the responses from the LLM. \begin{DoxyAuthor}{Author}
