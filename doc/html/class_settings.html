Detailed Description

The Settings class provides a dialog for configuring application settings.

This class allows the user to set API keys for LLM and audio services, as well as select a summary layout preference. It uses a singleton pattern to ensure that only one instance of the settings dialog is created.

Note
The settings are stored in a configuration file for persistence across sessions.
